PART II: HOW TO USE

1. Download the application
2. Set all the settings according to PART V
3. Run App_Name.exe file

PART III: MAIN MENU

1. ADD A USER
 1.1. This will allow you to add a new user to the database by filling a form
 1.2. At the end of the form press Y to submit the form and the new user will be added to the database
2. EDIT A USER
 2.1. This will allow you to edit an existing user by filling a form
 2.2. At the end of the form press Y to submit the form and the user information will be edited
3. DELETE A USER
 3.1. This will allow you to delete an existing user
 3.2. Insert a valid user ID number to delete the user
4. EXPORT
 4.1. This function allowes you to save your current database as a csv file
5. IMPORT
 5.1. This function allowes you to load users to your database from a csv file
 5.2. Warning! All current users will be deleted
6. LOAD FOLDER
 6.1. Loads a folder location
7. EDIT SCRIPT
 7.1. This will allow you to edit the script file
8. RUN SCRIPT
 8.1. This function activates the script file

PART IV: SCRIPT

1. SUM(a, b)
 1.1. a - Integer number
 1.2. b - Integer number
 1.3. Return Value: a + b
2. AVG(list_of_numbers)
 2.1. list_of_numbers - A list of numbers in the following format [x1 x2 x3 x4 ...]
 2.2. Return Value: Sccssesful import True or False
3. HELLO_WORLD()
 3.1. Return Value: Prints HELLO WORLD

PART V: SETTINGS

1. Database location --> The location of the user database. For example: database.db
